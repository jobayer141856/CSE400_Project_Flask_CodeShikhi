C Operations

একটি অপারেটর একটি প্রতীক যা একটি মান বা একটি ভেরিয়েবলের উপর কাজ করে।একটি গাণিতিক অপারেটর সংখ্যাসূচক মান (ধ্রুবক এবং চলক) এর উপর গাণিতিক ক্রিয়াকলাপ যেমন যোগ, বিয়োগ, গুণ, ভাগ ইত্যাদি করে থাকে।

নীচের উদাহরণে, আমরা দুটি মান একসাথে যোগ করার জন্য + অপারেটর ব্যবহার করি:

Examples

int sum1 = 100 + 50;      // 150(100+50)
int sum2 =sum1 + 250;      // 400(150+250)
int sum3 =sum1+sum2;      // 800(400+400)

সি অপারেটরদের নিম্নলিখিত গ্রুপে বিভক্ত করে:

  • Arithmetic operators
  • Assignment operators
  • Comparison operators
  • Logical operators
  • Bitwise operators

Arithmetic Operators

পাটিগণিত অপারেটরগুলি সাধারণ গাণিতিক ক্রিয়াকলাপ সম্পাদন করতে ব্যবহৃত হয়।

Operation Name Description Example
+ additon দুটি মান একসাথে যোগ করে x + y
- Subtraction একটি মান থেকে আরেকটি মান বিয়োগ করে x - y
* Multiplication দুটি মান গুণ করে x * y
/ Division একটি মানকে আরেকটি দিয়ে ভাগ করে x / y
% Modulus বিভাজন অবশিষ্ট ফেরত দেয় x % y
++ Increment একটি ভেরিয়েবলের মান 1 দ্বারা বৃদ্ধি করে x ++
-- Decrement একটি ভেরিয়েবলের মান 1 দ্বারা হ্রাস করে x --

Assignment Operators

অ্যাসাইনমেন্ট অপারেটরগুলি ভেরিয়েবলের মান নির্ধারণ করতে ব্যবহৃত হয়।

সমস্ত অ্যাসাইনমেন্ট অপারেটরদের একটি তালিকা:

Operation Example Same As
= X = 5 X = 5
+= X += 5 X = X + 5
-= X -= 5 X = X - 5
*= X *= 5 X = X * 5
/= X /= 5 X = X / 5
%= X %= 5 X = X % 5
&= X &= 5 X = X & 5
|= X |= 5 X = X | 5
^= X ^= 5 X = X ^ 5

Comparison Operators

তুলনা অপারেটর দুটি মান (বা ভেরিয়েবল) তুলনা করতে ব্যবহৃত হয়। এটি প্রোগ্রামিংয়ের ক্ষেত্রে গুরুত্বপূর্ণ, কারণ এটি আমাদের উত্তর খুঁজে পেতে এবং সিদ্ধান্ত নিতে সাহায্য করে।

একটি তুলনার রিটার্ন মান হয় 1 বা 0, যার অর্থ সত্য (1) বা মিথ্যা (0)। এই মানগুলি বুলিয়ান মান হিসাবে পরিচিত, এবং আপনি বুলিয়ান এবং If..Else অধ্যায়ে সেগুলি সম্পর্কে আরও শিখবেন।

সমস্ত তুলনা অপারেটরের একটি তালিকা:

Operation Name Example
== Equal to X == Y
!= Not Equal X != Y
> Greater than X > Y
< Less than X < Y
>= Greater than or equal X >= Y
<= Less than or Equal X <= Y

Logical Operators

আপনি লজিক্যাল অপারেটরদের সাথে সত্য বা মিথ্যা মান পরীক্ষা করতে পারেন।

লজিক্যাল অপারেটরগুলি ভেরিয়েবল বা মানগুলির মধ্যে যুক্তি নির্ধারণ করতে ব্যবহৃত হয়:

Operator Name Description Example
&& Logical and উভয় বিবৃতি সত্য হলে সত্য প্রদান করে x < 5 && x < 10
|| Logical or বিবৃতিগুলির একটি সত্য হলে সত্য প্রদান করে x < 5 || x < 10
! Logical not ফলাফল বিপরীত, ফলাফল সত্য হলে মিথ্যা প্রদান করে !(x < 5 && x < 10)

Exercise

10 কে 5 দিয়ে গুণ করতে শূন্যস্থান পূরণ করুন এবং ফলাফলটি প্রিন্ট করুন:

int x= 10 ;
int y= 5 ;
printf("", x y);
