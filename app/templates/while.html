While loop in C

সি-তে থাকা while লুপটি একটি পরীক্ষার শর্ত মূল্যায়ন করতে এবং শর্তটি সত্য না হওয়া পর্যন্ত লুপের বডিতে পুনরাবৃত্তি করতে ব্যবহৃত হয়। শর্তটি মিথ্যা হলে লুপ শেষ হয়। এই লুপটিকে একটি প্রাক-পরীক্ষিত লুপ হিসাবেও পরিচিত কারণ এটি সাধারণত ব্যবহৃত হয় যখন পুনরাবৃত্তির সংখ্যা ব্যবহারকারীর কাছে সময়ের আগে অজানা থাকে।


একটি নির্দিষ্ট শর্ত সত্য না হওয়া পর্যন্ত যখন লুপ কোডের একটি ব্লকের মধ্য দিয়ে চলে:

Syntax

while (condition) {
// code block to be executed
}

নীচের উদাহরণে, লুপের কোডটি বারবার চলবে, যতক্ষণ না একটি ভেরিয়েবল (i) 5-এর কম হয়:

Example

int i = 0 ;
while (i < 5 ) {
printf( "%d\n", i);
i++;
}

বিঃদ্রঃ কন্ডিশনে (i++) ব্যবহৃত ভেরিয়েবল বাড়াতে ভুলবেন না, অন্যথায় লুপ শেষ হবে না!
