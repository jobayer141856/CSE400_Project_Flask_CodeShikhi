সি-তে ডেটাটাইপ কী?

ভেরিয়েবল যে ধরনের ডেটা সংরক্ষণ করতে পারে তা নির্দিষ্ট করে ডেটাটাইপ ,যেমন পূর্ণসংখ্যা, অক্ষর ইত্যাদি।

Example

//Create Variable

int myNum=5;       // intiger variable
float myFnum=10.5;       // floating point variable
char ch='M';       // character variable

// Print variables
printf("%d",myNum);
printf("%f",myFnum);
printf("%c",ch);

Basic data types

ডেটা টাইপ ভেরিয়েবল সংরক্ষণ করবে তথ্যের আকার এবং ধরন নির্দিষ্ট করে।

এই টিউটোরিয়ালে, আমরা সবচেয়ে মৌলিক বিষয়গুলিতে ফোকাস করব:
Datatype Size Description
int 2 or 4 bytes
দশমিক ছাড়া পূর্ণ সংখ্যা সংরক্ষণ করে
float 4 bytes
এক বা একাধিক দশমিক সমন্বিত ভগ্নাংশ সংখ্যা সংরক্ষণ করে। 6-7 দশমিক সংখ্যা সংরক্ষণের জন্য যথেষ্ট
double 8 bytes
এক বা একাধিক দশমিক সমন্বিত ভগ্নাংশ সংখ্যা সংরক্ষণ করে। 15 দশমিক সংখ্যা সংরক্ষণের জন্য যথেষ্ট
char 1 byte একটি একক অক্ষর/অক্ষর/সংখ্যা বা ASCII মান সঞ্চয় করে

Basic Format Specifiers


প্রতিটি ডাটা টাইপের জন্য আলাদা ফরম্যাট স্পেসিফায়ার আছে। এখানে তাদের কিছু:
Format Specifier Datatype
%d or%i int
%f float
%lf double
%c char

Set Decimal Precision

You have probably already noticed that if you print a floating point number, the output will show many digits after the decimal point:

Example

float myFnum = 3.5;
double mydoublenum = 19.5;

printf("%f\n", myFnum);    // Outputs 3.500000
printf("%lf\n", mydoublenum);    // Outputs 19.500000

If you want to remove the extra zeros (set decimal precision), you can use a dot (.) followed by a number that specifies how many digits that should be shown after the decimal point:

Example

float myFnum = 3.5;

printf("%f\n", myFnum);    // Default will show 6 digits after the decimal point
printf("%.1f\n", myFnum);    // this will show 1 digits after the decimal point
printf("%.2f\n", myFnum);    // this will show 2 digits after the decimal point
printf("%.4f\n", myFnum);    // this will show 4 digits after the decimal point
