C If ... Else

if-else বিবৃতি হল একটি সিদ্ধান্ত গ্রহণের বিবৃতি যা নির্দিষ্ট শর্তের (পরীক্ষার অভিব্যক্তি) উপর ভিত্তি করে কোডের অংশটি কার্যকর করা হবে কিনা তা নির্ধারণ করতে ব্যবহৃত হয়। যদি প্রদত্ত শর্ত সত্য হয়, তাহলে if ব্লকের ভিতরের কোডটি কার্যকর করা হয়, অন্যথায় else ব্লকের ভিতরের কোডটি কার্যকর করা হয়।

Conditions and If Statements

আমরা ইতিমধ্যে শিখেছি যে C গণিত থেকে স্বাভাবিক যৌক্তিক অবস্থাকে সমর্থন করে :

  • Less than: a < b
  • Less than or equal to: a <= b
  • Greater than: a >= b
  • Greater than or equal to: a < b
  • Equal to: a == b
  • Not equal to: a != b

আমরা বিভিন্ন সিদ্ধান্তের জন্য বিভিন্ন ক্রিয়া সম্পাদন করতে এই শর্তগুলি ব্যবহার করতে পারি.

C এর নিম্নলিখিত শর্তসাপেক্ষ বিবৃতি রয়েছে:

  • Use if to specify a block of code to be executed, if a specified condition is true
  • Use else to specify a block of code to be executed, if the condition is false
  • Use else if to specify a new condition to test, if a specified condition is false
  • Use switch to specify many alternative blocks of code to be executed

Syntax

if(condition){
// block of code to be executed
}

আমরা শর্তে পরিবর্তনশীল এবং আদিম ডেটা ব্যবহার করতে পারি:

Syntax

if(condition1){
// block of code to be executed
}else if(condition2){
// block of code to be executed if the condition1 is false and condition2 is true
}else{
// block of code to be executed if the condition1 is false and condition2 is false
}

Example

int time = 22;
if (time < 10 ) {
printf("Good morning.");
} else if (time < 20 ) {
printf("Good day.");
} else {
printf("Good Evening");
} // Outputs "Good evening."
