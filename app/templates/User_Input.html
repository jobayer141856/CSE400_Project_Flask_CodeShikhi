User Input

আপনি ইতিমধ্যে শিখেছেন যে printf() ব্যবহার করা হয় সি-তে মান আউটপুট করতে।

ব্যবহারকারীর ইনপুট পেতে, আপনি scanf() ফাংশন ব্যবহার করতে পারেন:


অক্ষর সংরক্ষণের জন্য স্ট্রিং ব্যবহার করা হয়।

উদাহরণস্বরূপ, "hello world" অক্ষরের একটি স্ট্রিং।

অন্যান্য অনেক প্রোগ্রামিং ল্যাঙ্গুয়েজ থেকে ভিন্ন, সহজে স্ট্রিং ভেরিয়েবল তৈরি করার জন্য সি-তে স্ট্রিং টাইপ নেই। পরিবর্তে, আপনাকে অবশ্যই char টাইপ ব্যবহার করতে হবে এবং সি-তে একটি স্ট্রিং তৈরি করতে অক্ষরের একটি অ্যারে তৈরি করতে হবে:

Example

ব্যবহারকারীর দ্বারা input করা একটি সংখ্যা আউটপুট:
// Create an integer variable that will store the number we get from the user
int myNum;

// Ask the user to type a number
printf("Type a number: \n");

// Get and save the number the user types
Scanf("%d",&myNum);

// Output the number the user typed
printf("Your number is :%d",myNum);

একাধিক ইনপুট

scanf() ফাংশন একাধিক ইনপুট (নিম্নলিখিত উদাহরণে একটি পূর্ণসংখ্যা এবং একটি অক্ষর) অনুমতি দেয়:

Example

// Create an int and a char variable
int myNum;
char myChar;

// // Ask the user to type a number AND a character
printf("Type a number AND a character and press enter: \n");

// Get and save the number AND character the user types
Scanf("%d %c",&myNum,&myChar);

// print the number
printf("Your number is :%d",myNum);

// print the character
printf("Your character is :%c",myChar);

Example
