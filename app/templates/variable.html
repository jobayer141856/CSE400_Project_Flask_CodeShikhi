ভেরিয়েবল কি?

প্রোগ্রামিং-এ, একটি ভেরিয়েবল হল এমন একটি মান যা পরিবর্তিত হতে পারে, শর্ত বা প্রোগ্রামে পাঠানো তথ্যের উপর নির্ভর করে। সাধারণত, একটি প্রোগ্রামে নির্দেশাবলী থাকে যা কম্পিউটারকে বলে যে কী করতে হবে এবং প্রোগ্রামটি যখন চলমান তখন ব্যবহার করে এমন ডেটা।

ছবিতে আমরা দেখতে পাচ্ছি যে টেক্সট বক্সের নাম পরিবর্তন হচ্ছে যেমন,আশিফ,লিন,ডেভিড। তাই ভেরিয়েবল একটি কন্টেইনারের মতো আমরা সেখানে কন্টেইনার ডেটাটাইপের উপর নির্ভর করে বিভিন্ন মান সংরক্ষণ করতে পারি।


সি-তে, বিভিন্ন ধরনের ভেরিয়েবল রয়েছে (বিভিন্ন কীওয়ার্ড দিয়ে সংজ্ঞায়িত), উদাহরণস্বরূপ:

  • int - stores integers (whole numbers), without decimals, such as 123 or -123
  • float - stores floating point numbers, with decimals, such as 19.99 or -19.99
  • Char - stores single characters, such as 'a' or 'B'. Char values are surrounded by single quotes

Declaring a Variable

একটি ভেরিয়েবল তৈরি করতে, ধরনটি নির্দিষ্ট করুন এবং একটি মান নির্ধারণ করুন:

Syntax

type variable = value

এখানে টাইপ হল ডেটা টাইপ যেমন int, long long, Float, Char এবং Equal সাইন পরিবর্তনশীল নামে মান নির্ধারণ করছে.


int এর একটি ভেরিয়েবল t_Num তৈরি করুন এবং এটিতে মান 10 বরাদ্দ করুন:

Example

int t_Num = 10;

More Examples

int s_Num = 100;       // intiger variable

char c = 'a';       // character variable

float f_Num = 10.55;       // floating point variable


কিভাবে একটি ভেরিয়েবল আউটপুট দেখাতে হয়,উদাহরণস্বরূপ:

Output Examples

printf("%d",s_Num);

printf("%c",c);

printf("%f",f_Num);

Exercise

myNum নামে একটি ভেরিয়েবল তৈরি করুন এবং এটিতে মান 50 বরাদ্দ করুন।
