Arrays

অ্যারেগুলি প্রতিটি মানের জন্য পৃথক ভেরিয়েবল ঘোষণা করার পরিবর্তে একটি একক ভেরিয়েবলে একাধিক মান সংরক্ষণ করতে ব্যবহৃত হয়।।


যখন আপনি জানেন যে আপনি ঠিক কতবার কোড ব্লকের মাধ্যমে লুপ করতে চান, একটি while লুপের পরিবর্তে for loop ব্যবহার করুন:

একটি অ্যারে তৈরি করতে, ডেটা টাইপ (যেমন int) সংজ্ঞায়িত করুন এবং বর্গাকার বন্ধনী [] দ্বারা অ্যারের নাম উল্লেখ করুন।

এটিতে মান সন্নিবেশ করতে, কোঁকড়া ধনুর্বন্ধনীর ভিতরে একটি কমা দ্বারা পৃথক তালিকা ব্যবহার করুন:

Example

int myNumbers[] = { 25, 50, 75, 100 };


আমরা এখন একটি ভেরিয়েবল তৈরি করেছি যেখানে চারটি পূর্ণসংখ্যার অ্যারে রয়েছে।


একটি অ্যারের উপাদান অ্যাক্সেস


একটি অ্যারে উপাদান অ্যাক্সেস করতে, এর সূচক (index) নম্বর ব্যবহার করো ।

অ্যারে ইনডেক্স 0 দিয়ে শুরু হয়: [0] প্রথম উপাদান। [1] দ্বিতীয় উপাদান, ইত্যাদি |

এই বিবৃতিটি myNumbers-এ প্রথম উপাদান [0]-এর মান অ্যাক্সেস করে:

Example

int myNumbers[] = { 25, 50, 75, 100 };
printf( "%d", myNumbers[0]);

// Outputs 25

একটি অ্যারে উপাদান পরিবর্তন করুন


একটি নির্দিষ্ট উপাদানের মান পরিবর্তন করতে, সূচক নম্বর ব্যবহার করো :

Example

myNumbers[0] = 33;

Example

int myNumbers[] = { 25, 50, 75, 100 };
myNumbers[0] = 33;
printf( "%d", myNumbers[0]);

// Outputs 33 instead of 25

Loop Through an Array


আপনি ফর লুপ দিয়ে অ্যারের উপাদানগুলি অ্যাক্সেস করতে পারেন।

নিম্নলিখিত উদাহরণটি myNumbers অ্যারের সমস্ত উপাদানকে আউটপুট করে:

Example

myNumbers[] = { 25, 50, 75, 100 };
int i;
for (i = 0; i < 4 ; i++) {
printf( "%d", myNumbers[i]);
}

অ্যারে সাইজ সেট করুন

অ্যারে তৈরি করার আরেকটি সাধারণ উপায় হল অ্যারের আকার নির্দিষ্ট করা এবং পরে উপাদান যোগ করা:

Example

//Declare an array of four integer
int numbers [4];

//Add element
myNumbers[0] = 25;
myNumbers[1] = 50;
myNumbers[2] = 75;
myNumbers[3] = 100;

Exercise
