Strings

সি প্রোগ্রামিং-এ একটি স্ট্রিং হল একটি শূন্য অক্ষর '\0' দিয়ে সমাপ্ত অক্ষরের একটি ক্রম। সি স্ট্রিং অক্ষরগুলির একটি অ্যারে হিসাবে সংরক্ষণ করা হয়। একটি অক্ষর অ্যারে এবং একটি সি স্ট্রিংয়ের মধ্যে পার্থক্য হল যে সি-তে স্ট্রিংটি একটি অনন্য অক্ষর '\0' দিয়ে শেষ করা হয়।।




অক্ষর সংরক্ষণের জন্য স্ট্রিং ব্যবহার করা হয়।

উদাহরণস্বরূপ, "hello world" অক্ষরের একটি স্ট্রিং।

অন্যান্য অনেক প্রোগ্রামিং ল্যাঙ্গুয়েজ থেকে ভিন্ন, সহজে স্ট্রিং ভেরিয়েবল তৈরি করার জন্য সি-তে স্ট্রিং টাইপ নেই। পরিবর্তে, আপনাকে অবশ্যই char টাইপ ব্যবহার করতে হবে এবং সি-তে একটি স্ট্রিং তৈরি করতে অক্ষরের একটি অ্যারে তৈরি করতে হবে:

Example

char greetings[] = "Hello world!";


মনে রাখবেন যে আপনাকে ডবল উদ্ধৃতি ("") ব্যবহার করতে হবে।

স্ট্রিং আউটপুট করতে, আপনি printf() ফাংশনটি ফরম্যাট স্পেসিফায়ার %s এর সাথে ব্যবহার করতে পারেন সি কে জানাতে যে আমরা এখন স্ট্রিংগুলির সাথে কাজ করছি:


Example

char greetings[] = "hello world" ;
printf( "%s", greetings);

অ্যাক্সেস স্ট্রিং


যেহেতু স্ট্রিংগুলি আসলে সি-তে অ্যারে, তাই আপনি বর্গাকার বন্ধনী [] এর ভিতরে তার সূচক নম্বর উল্লেখ করে একটি স্ট্রিং অ্যাক্সেস করতে পারেন:

এই উদাহরণটি String এর প্রথম অক্ষর (0) প্রিন্ট করে:

Example

char greetings[] = "hello world" ;
printf( "%c", greetings[0]);

স্ট্রিং পরিবর্তন করুন

একটি স্ট্রিং-এ একটি নির্দিষ্ট অক্ষরের মান পরিবর্তন করতে, index নম্বর পড়ুন এবং ' ' উদ্ধৃতি ব্যবহার করুন:

Example

char greetings[] = "hello world";
greetings[0] = 'j';
printf( "%s", greetings[0]);

// Outputs Jello World! instead of Hello World!
